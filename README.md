---# 🎯 SNS Talent Pool

**学生×企業をつなぐ、次世代SNS採用プラットフォーム**

## 📖 概要

SNS Talent Poolは、TikTok・Instagram・X(Twitter)などのSNSを活用する学生と、優秀な人材を求める企業・学生団体をマッチングする管理システムです。

従来の履歴書ベースの採用ではなく、**SNSでの活動実績**や**スキル**を可視化し、企業が求める人材を効率的に見つけることができます。

---

## 🎯 なぜこのシステムを作ったか

### 解決したい課題

**学生側の課題:**
- 自分のSNS活動が評価されない
- インターン・学生団体の情報が散在している
- 自分に合った機会を見つけるのが難しい

**企業側の課題:**
- SNSマーケティングができる学生を見つけにくい
- 採用コストが高い
- 学生の実際のスキルが見えにくい

### このシステムの価値

✅ **SNS実績の可視化** - TikTok、Instagram、Xのアカウントを一元管理  
✅ **スキルベースマッチング** - 興味・スキルで最適な学生を自動マッチング  
✅ **効率的な採用** - フィルタ機能で条件に合う学生をすぐに発見  
✅ **データ管理** - 学生情報を一元管理し、いつでもアクセス可能

## ✨ 主な機能

### 1. 学生情報管理
- 基本情報(名前、大学、学年)
- 興味分野・スキルのタグ管理
- SNSアカウント連携(Instagram/TikTok/X)
- 自己PR表示

### 2. 高度なフィルタリング
- **興味分野**で絞り込み(マーケティング、エンジニアリング、デザインなど)
- **スキル**で絞り込み(SNS運用、動画編集、プログラミングなど)
- **大学**で絞り込み
- **キーワード検索**(名前、大学、スキルから横断検索)

### 3. マッチングスコア
- 学生の興味・スキル・SNSアカウント数から自動算出
- 80%以上の学生を「マッチング済み」として表示
- 企業のニーズに合った学生を優先表示

### 4. 詳細モーダル
- 学生の詳細情報をワンクリックで表示
- メール連絡ボタン(mailto自動生成)
- データ出力機能(JSON形式)

### 5. レスポンシブデザイン
- PC・タブレット・スマホ対応
- グリッド表示/リスト表示の切り替え
- モダンで視認性の高いUI

## 🚀 セットアップ方法

### 必要なもの
- Webブラウザ(Chrome, Safari, Firefox推奨)
- テキストエディタ(VSCode推奨)

### インストール手順

1. **リポジトリをクローン**
```bash
git clone https://github.com/yukki-88/sns-talent-pool.git
cd sns-talent-pool
```

2. **ファイル構成を確認**
```
sns-talent-pool/
├── index.html       # メインページ
├── style.css        # スタイル
├── script.js        # 機能
├── data.json        # 学生データ
└── README.md        # このファイル
```

3. **ブラウザで開く**
- `index.html` をダブルクリック
- または、Live Server(VSCode拡張機能)で開く

## 📖 使い方

### 基本操作

1. **学生一覧を見る**
   - ページを開くと、登録されている学生が自動表示されます

2. **フィルタで絞り込む**
   - 興味分野・スキル・大学のドロップダウンから選択
   - キーワード検索ボックスで自由検索
   - 「フィルターをリセット」で全学生を再表示

3. **学生の詳細を見る**
   - 学生カードをクリックすると詳細モーダルが開きます
   - 連絡先、SNSアカウント、自己PRを確認できます

4. **学生に連絡する**
   - 「メールで連絡」ボタンでメールアプリが起動
   - 「データ出力」ボタンで学生情報をJSON形式でダウンロード

5. **表示を切り替え**
   - 「グリッド」「リスト」ボタンで表示形式を変更

## 📊 データ構造

### data.json の形式
```json
{
  "id": 1,
  "name": "田中 太郎",
  "email": "tanaka@example.com",
  "phone": "090-1234-5678",
  "university": "早稲田大学",
  "grade": "3年生",
  "interests": ["マーケティング", "企画"],
  "skills": ["SNS運用", "動画編集"],
  "sns": {
    "instagram": "https://instagram.com/...",
    "tiktok": "https://tiktok.com/@...",
    "twitter": "https://twitter.com/..."
  },
  "bio": "自己PR文..."
}
```

### 学生データの追加方法

`data.json` に上記形式でデータを追加すると、自動的に画面に反映されます。
